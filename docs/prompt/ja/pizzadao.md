# 今決まっている要件

## 画面

### 画面数

1 dayハッカソンなので1画面で完結させます。

### 画面構成

- ヘッダー部分：WalletConnect (OnChainKitで提供されるものを使う)
- メイン部分： 
  - ルーレット部分
    - ダーツの的のように回転する丸いボードがある
    - ゲームスタートボタンを押すとボードが回る
    - ユーザーはボードをタップする
    - タップした部分のピザ切れが埋まる
    - タップで埋まるピザの味は6種類くらいからランダム
    - 同じ色のピザをできる限り揃えるとポイントが増える
      - 一切れ12個
      - 隣同士で同じ味だとポイントアップ
      - 全部同じ色だと最高にポイントが高い
      - ポイントに応じてミントできるNFTのランクを分ける
        - ダイアモンドランク
        - ゴールドランク
        - シルバーランク
        - ブロンズランク
    - 全部埋まったら最終的なスコアに応じてランクを表示しミントボタンも表示する
    - ミントしたらもう一度リセットでゲーム開始
